# firebase-user-auth
A template for user authentication using Firebase

# Installation
Git clone
```
https://github.com/yourusername/firebase-user-auth.git
```
or
download zip folder

Change directory if haven't
```
cd firebase-user-auth
```

Install packages
```
npm install
```

Run the project
```
npm start
```
